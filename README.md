# LocalStream
A Smart TV application designed to deliver local news and live streaming content to the living room. It will feature a dynamic home screen with categorized content rows (VODs), a live broadcast player, and a detail view for individual video content.

## Tech Stack
- **Core**: React Native, Expo, TypeScript
- **State Management**: Zustand
- **Data Fetching**: React Query
- **Styling & Animation**: React Native Reanimated
- **Libraries & APIs**: React Native TV OS
- **AI**: Google Gemini

## The Why
I am building this app to get experience with making TV apps using React Native. I am using Google Gemini as a manager that gives me tickets to complete tasks. Once a task is completed, I make a pull request to GitHub.
